# **App Name**: Fractal Links

## Core Features:

- Link Display: Display a list of links with labels and subtitles, as defined in the data structure.
- Customizable Links: Store link data in a Firestore database for easy editing without code deployment.
- External Navigation: Allow users to navigate to external websites via the links provided.
- Styled Components: Use Tailwind CSS for styling to create a visually appealing and responsive design.

## Style Guidelines:

- Primary color: Deep magenta (#FF2BA6) to capture the vibrant and modern essence of Fractal Studio Lab.
- Background color: Very dark desaturated blue (#050112) to provide a sophisticated backdrop that makes the pink buttons stand out.
- Accent color: Soft purple (#3A7BFF) to complement the magenta and add depth.
- Body and headline font: 'Poppins' (sans-serif) for a contemporary and precise feel.
- Simple, clean icons to represent different services.
- Single-page layout with a central card for focused user experience.
- Subtle hover animations on the buttons to enhance interactivity.
